# MovieSearch

### Movie Search and Filter operations are performed on data obtained from TMDB API.
This was a selection criteria task for ReactJS Internship at IIT-B

### Requirements: <br>
- Build a React app that allows users to search for movies and view details about them. <br>
- Use an API to retrieve movie data and display it in a user-friendly format. <br>
- Implement the following extra features:<br>
  - Search functionality to search for movies by title<br>
  - The movie display format should be presented in a card layout.<br>
  - Filter functionality to filter movies by rating, most voted, or date released<br>
  - A responsive UI that adapts to different screen sizes<br>
          
### The API Used:
- TMDB API was used to perform operations. <br>
